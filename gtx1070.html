  • arrow_drop_downProduct Review

    Product Review

    Advantages
    • Powerful enough to run games at 1440p
    • Cooling is quiet yet effective
    • Supports the latest Displayport
    • able to run the latest games at high - ultra (maxed) settings
    • fans don't start until a specific temp is reached

    Disadvantages
    • Overlocking is heavily reliant on memory type (Micron, or Samsung)
    • Won't fit in small cases due to how wide it is
    • Does not support RGB
    • Software is not as good as MSI afterburner



    Conclusion
    Zotac’s flagship 1070 the AMP extreme edition is a mammoth of a card packs a heavy punch for a cheap price. This card can easily run the latest games maxed out with a resolution of 1080p 60hz at 60+fps, and 1440p at high-ultra setting with around 50+fps. The AMP extreme also provides some of the best cooling on the market with triple icestorm fans which don’t turn on till 50+ degrees Celsius. This allows for a quiet and enjoyable gaming experience. However, this card is a gamble due to the micron / Samsung lottery. If the card contains micron memory, the overclocking capabilities will be severely reduced to almost not being able to overclock at all. This is exactly what happened with the card we reviewed. But straight out the box not overclocked this card performs perfectly providing a stable and enjoyable experience. As a conclusion I would still recommend this card if you aren’t looking to overclock. However, for overclocking I would chose a different model such as the MSI GTX 1070 gaming.

    Final Score

    8/10
